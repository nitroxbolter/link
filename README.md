# Links Importantes

Site simples para guardar links importantes, hospedado no GitHub Pages.

## Estrutura

- **Home**: Página inicial com informações sobre o site
- **Bases**: Links relacionados a bases de conhecimento
- **Sprites**: Links de recursos gráficos e sprites
- **GitHub**: Links importantes do GitHub

## Como usar

1. Edite o arquivo `index.html` para adicionar seus links nas seções correspondentes
2. Faça commit e push para o GitHub
3. Ative o GitHub Pages nas configurações do repositório

## Personalização

Você pode personalizar:
- Cores no arquivo `styles.css`
- Links nas seções do `index.html`
- Ícones e textos no menu lateral
